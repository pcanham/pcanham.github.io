Seeing emails with the following subject in adm or roots mail file ***UPDATE /etc/acct/holidays WITH NEW HOLIDAYS***

July 17, 2008

To get rid of these edit the following file and change the year to the current year

File: /etc/acct/holidays

Example Output below with section marked as bold what you need to alter

  • @(#)holidays  January 1, 2006 *
  • Prime/Nonprime Table for UNIX Accounting System *
  • Curr  Prime   Non-Prime
  • Year  Start   Start * **2008 ** 0800    1800 *
  • only the first column (month/day) is significiant. *
  • month/day     Company *               Holiday * 1/1             New Years Day 7/4             Indep. Day 12/25           Christmas
