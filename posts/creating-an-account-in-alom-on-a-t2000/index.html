Creating an account in ALOM on a T2000

November 26, 2007

  1. Get into ALOM (from console for ‘#.’ minus quotes)

  2. to add the user run the following ‘useradd ‘

  3. To assign a password this account run ‘userpassword ‘

  4. Assigning permissions to the user account ‘userperm [a|u|c|r]’ e.g. full permissions would be ‘userperm aucr’

Permission Level Description
a Administrative. This user is authorized to change the state of ALOM configuration variables and reboot ALOM. u User administration. This user is authorized to add users and delete users, change user permissions, and change the authorization level of other users.
c Console permission. This user is authorized to connect to the host server system console.
r Reset and power permission. This user is authorized to reset the host server, and power the server on and off.
