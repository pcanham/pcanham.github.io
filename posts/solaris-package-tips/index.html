Solaris Package Tips

October 23, 2007

View the files that make up a Solaris package

/usr/sbin/pkgchk -l SUNWcsr | grep Pathname | sed ’s/Pathname: (.*)/1/’

Check which package a file belongs to

/usr/sbin/pkgchk -lp somefile
