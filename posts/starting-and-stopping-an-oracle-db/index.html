Starting and stopping an oracle DB

Databases · Oracle
Databases · Oracle

STARTING DATABASE

  1. Start Listerner up 
       
   # lsnrctl start

1. Connect to sqlplus as sysdba
   SQL> connect sys as sysdba

2. run command 'startup'
   SQL> startup

3. Exit from sqlplus
   SQL> exit
