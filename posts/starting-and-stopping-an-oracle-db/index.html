Starting and stopping an oracle DB

March 3, 2009

STARTING DATABASE

  1. Start Listerner up

    lsnrctl start

  2. Connect to sqlplus as sysdba SQL> connect sys as sysdba

  3. run command ‘startup’ SQL> startup

  4. Exit from sqlplus SQL> exit

STARTING Enterprise Manager

  1. Login as oracle user

  2. run following command

    emctl start dbconsole

example

-bash-3.00$ emctl start dbconsole Oracle Enterprise Manager 10g Database Control Release 10.2.0.2.0 Copyright © 1996, 2005 Oracle Corporation.  All rights reserved. http://dbman.toon-net.local:5500/em/console/aboutApplication

Starting Oracle Enterprise Manager 10g Database Control ………….. started.

Logs are generated in directory /db/oracle/102/dbman.toon-net.local_cava/sysman/log

Starting iSQL*Plus (http://dbman.toon-net.local:5560/isqlplus/)

  1. Login as oracle

  2. run following command

    isqlplusctl start

example

-bash-3.00$ isqlplusctl start iSQL*Plus 10.2.0.2.0 Copyright © 2003, 2005, Oracle.  All Rights Reserved. Starting iSQL*Plus … iSQL*Plus started.

SHUTTING DOWN DATABASE

  1. Connect to sqlplus as sysdba connect sys as sysdba
  2. run command ‘shutdown immediate’ SQL> shutdown immediate
  3. Exit from sqlplus SQL> exit

example
