Enabling SNMP (SMA - System Management Agent) under Solaris 10

February 25, 2009

Installation

Install SUNWsmaS, SUNWsmagt, SUNWsmcmd, SUNWsmdoc, SUNWsmmgr

Configuration

cd /etc/sma/snmp cp snmpd.conf cp snmpd.conf snmpd.conf.bak-date +%F vi snmpd.conf change rocommunity (by default set to public) Change syslocation and syscontact to something useful e.g.

From

syslocation  “System administrators office” syscontact  “System administrator”

To

syslocation  “Test x86 Server on Floor 2” syscontact  “monitoring@example.com”

Start or restart SMA service

svcadm restart svc:/application/management/sma:default # Restart svcadm enable svc:/application/management/sma:default # Start

Test from remote server

pcanham@jafar:~$ snmpwalk -v 2c -c public 192.168.2.9 sysDescr.0 SNMPv2-MIB::sysDescr.0 = STRING: SunOS dbman.work.toon-net.local 5.10 Generic_127112-11 i86pc
