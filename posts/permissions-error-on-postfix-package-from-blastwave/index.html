Permissions error on postfix package from blastwave

January 27, 2009

Error - Can send emails as root but not any other user

Error

May 20 23:32:51 server.example.com postfix/postdrop[7779]: [ID 947731 mail.warning] warning: mail_queue_enter: create file maildrop/610017.7779: Permission denied

Solution

root@server:/> cd /etc/opt/csw/postfix
root@server:/etc/opt/csw/postfix> ./post-install set-permissions command_directory=/opt/csw/sbin
