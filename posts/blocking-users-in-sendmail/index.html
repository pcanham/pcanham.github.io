Blocking Users in Sendmail

MTA
Sendmail

Instructions

  1. Go to ‘/etc/mail’
  2. Put a line in ‘/etc/mail/access’ along the lines of ‘email address DISCARD’
  3. Update table ‘makemap hash access.db < access’

Example

  1. Browse to folder 

       cd /etc/mail

  2. Edit access vi access 

       # joebloggs@example.com - email address being blocked

   # due to being invalid

   joebloggs@example.com               DISCARD

  3. Update table 

       makemap hash access.db < access
