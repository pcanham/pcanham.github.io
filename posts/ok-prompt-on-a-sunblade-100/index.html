OK prompt on a SunBlade 100

February 18, 2008

If you need to get into the OK prompt and you are physically at the machine do the following.

  1. Power off machine

  2. Power machine on

  3. When you hear the initial beep from the system highlighting it is starting up press the power supply button twice.
