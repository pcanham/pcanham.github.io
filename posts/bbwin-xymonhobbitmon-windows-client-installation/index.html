bbwin - xymon/hobbitmon windows client installation

February 10, 2012
bbwin hobbitmon xymon

BBwin installation

  1. Install bbwin client to machine

  2. Edit c:program filesBBWinetcBBWin.cfg telling it the server is “10.10.10.10” (will be under a setting called bbdisplay)

  3. edit the bbwin registry to make it use the FQDN and not the server name. Change the hostname entry from “%COMPUTERNAME%” to “%COMPUTERNAME%.%USERDNSDOMAIN%” minus the quotes

    • 64bit Machines = “HKEY_LOCAL_MACHINESOFTWAREWow6432NodeBBWin”.

    • 32bit Machines = “HKEY_LOCAL_MACHINESOFTWAREBBWin”

  4. With 64bit Windows you need to add an extra setting in bbwin.cfg just under the bbdisplay setting

  5. Start/Restart the “Big Brother Hobbit Client” (shortname for it if your running this though the net command is bbwin)

Software: http://sourceforge.net/projects/bbwin/

BBwin GUI

There is a configuration app you can run on the client’s (machines you wish to monitor) to make things easier, as of writing this page i have not tested this so cannot gurentee the results from it.

You can download from the following site https://sourceforge.net/projects/alertonfailure/files/MAG%20Server/BBWin_GUI_1.1.exe/download and more information can be found on it from http://www.squidworks.net/2011/08/xymon-new-bbwin-client-gui-1-0-released/
